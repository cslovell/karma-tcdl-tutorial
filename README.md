# karma-tcdl-tutorial
Files for the Karma tutorial at TCDL, Texas Conference on Digital Libraries

Starting Karma:

1. Open a CMD prompt from the start menu
2. cd c:\
3. cd Local
4. cd Web-Karma-master
5. cd karma-web
6. mvn jetty:run

Open Chrome:

1. Go to `localhost:8080`

Follow the instructions in the Wiki.
